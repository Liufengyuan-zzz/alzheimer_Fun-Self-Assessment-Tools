# Alzheimer's Disease Risk Self-Assessment Tool

This tool is a lightweight, interactive web application designed for the early screening and risk assessment of Alzheimer's disease. Based on the clinical principles of the **Mini-Cog** and **AD-8** assessment tools, it has been gamified, simplified, and optimized for self-testing by middle-aged and elderly individuals.

## 🚀 Features

- **Zero Dependencies**: Entirely built with vanilla HTML, CSS, and JavaScript. No external libraries, frameworks, or databases are required.
- **Dynamic & Randomized Question Bank**: Includes a curated repository of 16 distinct test questions divided into 8 mandatory assessment categories (e.g., Image Naming, Delayed Recall, Visuospatial Execution, Verbal Fluency, Orientation). In every test session, 8 questions are randomly drawn to prevent memory bias, while strictly preserving crucial clinical links (such as matching memory input with delayed extraction).
- **High-Performance Vector Graphics**: Images and visuals are fully embedded using heavily-optimized, hand-crafted inline SVGs and native emojis. This completely eliminates dependency on external image assets, guaranteeing instantaneous load times, infinite scaling without quality loss, and 100% offline usability.
- **Automated Scoring System**: Completes the multi-dimensional cognitive capability evaluation within 2-3 minutes.
- **Intuitive Feedback**: Interprets the final score (out of 24) and places the user into an intuitive three-tier color-coded system (Green: Low Risk, Yellow: Medium Risk, Red: High Risk), accompanied by actionable health advice.

## 💻 Tech Stack

- **HTML5**: semantic structural layouts.
- **CSS3**: responsive UI design boasting a purple-themed minimalist aesthetic (`#9b59b6`), CSS-based progress bars, modern card grids, and smooth animations.
- **JavaScript (ES6)**: handles test state, randomized variant logic, countdown timers (for fluency tests), and algorithmic scoring execution.

## 📖 How to Run

1. Navigate to the project directory.
2. Double click or open `alzheimer_test.html` in any modern web browser (Google Chrome, Safari, Edge, or Firefox).
3. Click "Start Test" on the landing page and sequentially complete the 8 mini-tasks.
4. Read your test results and corresponding health risk advice upon completion.
